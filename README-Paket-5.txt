TAXI ERBAS – PAKET 5: FUHRPARK-COUNTER

NEU:
- Die Fuhrpark-Zahlen zählen beim Scrollen von 0 bis zum Zielwert hoch.
- Die Animation startet erst, wenn der Fuhrpark-Bereich sichtbar ist.
- Sie läuft pro Seitenaufruf nur einmal.
- Das vorhandene Layout wurde nicht verändert.
- Bei aktivierter Einstellung „Bewegung reduzieren“ werden die Zielwerte direkt angezeigt.

INSTALLATION:
1. ZIP-Datei entpacken.
2. Alle Dateien in das GitHub-Repository hochladen.
3. Vorhandene Dateien ersetzen.
4. Seite nach dem GitHub-Pages-Update vollständig neu laden.
