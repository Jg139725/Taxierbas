PAKET 6 – PREMIUM-ANIMATIONEN

Dieses Paket verändert NICHT dein Layout.

Enthalten:
- Fade-Up beim Scrollen
- sanfte Hover-Effekte für Buttons
- leichter Zoom auf Bildern
- Karten erscheinen elegant
- berücksichtigt 'Bewegung reduzieren'

Einbau:
1. Öffne assets/js/main.js
2. Füge den Inhalt aus premium-animationen.js GANZ AM ENDE ein.
3. Speichern, GitHub hochladen.
